📚 API Documentation

Complete reference for all available endpoints

Core Endpoints

POST /api/v1/generate-content
Generate complete campaign with strategy, calendar, and posts
GET /health
Check API server health and service availability

Post Management

PUT /api/v1/posts/:postId
Edit post caption, hashtags, or call-to-action
POST /api/v1/posts/:postId/regenerate
Regenerate post caption and/or image using AI
DELETE /api/v1/posts/:postId
Permanently delete a post from database
POST /api/v1/campaigns/:campaignId/posts/add
Add an extra post for a specific date

Cache Management

GET /api/v1/cache/stats
View cache statistics (hits, misses, hit rate, size)
POST /api/v1/cache/clear
Clear all cache or specific pattern
